# Checklist Breve de Pruebas y Entrega

## 1) Checklist de pruebas mínimas
- [ ] Probé al menos 1 ejecución exitosa.
- [ ] Probé al menos 1 entrada inválida.
- [ ] Registré resultados en `docs/plan_de_pruebas.md`.

## 2) Checklist de documentación
- [ ] Completé `docs/propuesta.md`.
- [ ] Completé `docs/caso_de_uso.md`.
- [ ] Revisé `docs/estructura_repositorio.md`.
- [ ] Completé `docs/plan_de_pruebas.md`.

## 3) Checklist de ejecución
- [ ] `scripts/run.sh` corresponde a mi lenguaje elegido.
- [ ] El archivo principal existe en `src/`.
- [ ] La ejecución local funciona sin dependencias complejas.

## 4) Checklist de entrega final
- [ ] El proyecto mantiene alcance pequeño.
- [ ] No usé frameworks ni servicios externos.
- [ ] Mi propuesta es viable para el tiempo de la práctica.
- [ ] El repositorio está ordenado y entendible.
